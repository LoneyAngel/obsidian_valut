# 51单片机
- 与8051单片机一致的单片机都叫做51单片机
- 8位
- RAM： 512字节     运行内存
- ROM： 8K(Flash)  空间
- 工作频率：12MHz

![[Pasted image 20241215141326.png]]

- 数字使用16进制       ： c语言中的16进制的表示方法：0x开头
- 在keil中编写一个c语言的文件
- 然后在Stc中把对应的文件加载到单片机
- 使用stc的工具进行时间的程序

- 延时函数:  循环执行延时一秒的函数
- 暂时有点问题的延时函数
```c
#include <REGX52.H>
#include <INTRINS.H>

void Delay1ms(unsigned int time);

void main(){
	
	while(1){
		P2=0xFF;
		Delay1ms(500);
		P2=0xFD;
	}
		
}


void Delay1ms(unsigned int time)		//@12.000MHz
{
	while(time--){
		unsigned char i, j;
		i = 2;
		j = 239;
		do
		{
			while (--j);
		} while (--i);
	}
	
}
```
