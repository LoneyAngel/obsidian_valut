## scss
### 1. 安装依赖（以 Vite + Vue为例）
```bash
npm install sass --save-dev
```
### 2. 在组件中使用 SCSS（Vue 示例）

```vue
<style lang="scss">
$primary-color: #1890ff;
.button {
background-color: $primary-color;
&:hover {
background-color: darken($primary-color, 10%);
  }
}
</style>
```

📌 **结合使用建议：**

- SCSS 用于构建时的主题配置。
	- 主目录创建一个scss文件，然后需要的时候导入进行使用
- CSS 变量用于运行时动态主题切换。
